Hello!
This is our FBLA Quiz app. The app is a 20 question quiz.
The app is designed to test the knowledge of FBLA members by asking questions from different
FBLA topics such as competitive events, dates/locations, national officers, FBLA history.
Each quiz participant has 30 seconds to answer the question. If they answer early enough they can earn extra points.
This app was made with AndroidStudio using JAVA and XML. It can be run on a laptop using an
emulator or it can be run on an android phone/tablet.
